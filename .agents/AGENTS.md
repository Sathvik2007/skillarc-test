# SkillArc Design System & UI Rules

## 🎯 Design Philosophy
Every screen should follow one principle:
**"Content first, interface second."**
The UI should almost disappear, allowing courses, videos, assignments, and analytics to take center stage.
- No clutter.
- No unnecessary colors.
- No thick borders.
- No bulky buttons.
- Just elegant spacing, subtle animations, and premium typography.

---

## 🌈 Theme & Colors
The interface should feel like a bright, airy workspace.

### Background
Instead of plain white, use a soft mesh gradient background that is barely visible:
- Soft White
- Lavender
- Very Light Blue
- Mint Green
Blur everything together so it looks almost invisible.

### Color Palette
- **Primary**: Modern Indigo (`#6C63FF`) - Buttons, active menus, progress, links, charts.
- **Secondary**: Purple (`#8B5CF6`) - Hover states, AI features, accent gradients.
- **Accent**: Mint (`#00C2A8`) - Success, completed lessons, attendance, analytics.
- **Warning**: Gold/Orange (`#FFB020`) - Assignments, deadlines.
- **Error**: Red (`#F04438`) - Missing submissions, failed quizzes.
- **Background**: Soft Gray/Blue (`#F8FAFC`) - Never pure white.
- **Cards**: Pure White (`#FFFFFF`).
- **Heading Text**: Dark Gray (`#111827`).
- **Body Text**: Slate Gray (`#6B7280`).

---

## ✨ Glassmorphism (Glass 2.0)
Only floating elements should use glass.
- Navbar
- Notifications
- AI Assistant
- Command Palette
- Calendar Popup
- Profile Menu
- Modal Windows
Everything else stays solid.

---

## 🏠 Dashboard Layout
- **Greeting**: Personal and time-dependent (e.g. "Good Morning, Nikhil 👋 Let's continue building your future.")
- **Search Bar**: Centered at the top, large, rounded, glass effect, placeholder: "Search courses, faculty, students...", keyboard shortcut `⌘ K`.
- **Sidebar**: Very clean, white background, logo at top (S SkillArc in purple gradient).
- **Active Navigation Indicator**: Purple pill with a 4px indicator on the left.

---

## 🎭 Animations & Transitions
- Every interaction should feel alive.
- Cards gently lift on hover.
- Buttons slightly scale.
- Progress bars animate.
- Numbers count upward.
- Pages fade and slide.
- Modals softly blur the background.
- Loading uses skeleton placeholders—not spinners.
- Keep animations around 200ms (smooth, not distracting).

---

## 🔤 Typography
- **Headings**: `Plus Jakarta Sans` (bold, modern)
- **Body**: `Inter` (high readability)
- **Numbers**: `Space Grotesk` (clean analytics look)
